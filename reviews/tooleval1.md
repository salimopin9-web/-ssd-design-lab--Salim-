# UML Diagram Tool Evaluation
| Criterion | Tool A: diagrams.net | Tool B: PlantUML Web |
|---------|----------------------|---------------------|
| Learning curve | Very easy, drag-and-drop interface | Requires UML syntax, easier after basics |
| Version control friendliness | .drawio is XML-based; PNG for review | .puml is text-based and diff-friendly |
| Validation support | Manual visual checking | Built-in syntax validation |
| Collaboration | Easy to share via Drive/OneDrive | PR-friendly, easy code review |
| Export formats | PNG, SVG, PDF | PNG (regenerated from source) |
| Cost | Free | Free |
